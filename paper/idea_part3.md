## 创新点三：Issue Focus 驱动的多轮检索防跑偏机制

这是你非常有特色的地方，很多图检索论文不一定有。

可以写：

针对 Agent 多轮检索过程中 query 逐渐偏离初始 issue 的问题，本文设计 Issue Focus 查询增强机制，从初始 issue 和后续 query 中抽取结构化检索线索，并在后续检索中保留初始 issue 的约束信号，同时随轮次逐渐衰减其权重。该机制在保证任务相关性的同时，保留 Agent 局部探索的灵活性。

它相对前沿工作的特色是：

不是只做一次 query rewriting；
不是只做初始 localization；
而是服务于 Agent 多轮交互过程中的持续检索约束。

这个点一定要写进贡献。
